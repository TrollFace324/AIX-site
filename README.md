# AIX - Команда по робототехнике

Современный веб-сайт команды AIX по робототехнике.

## Структура проекта

```
AIX-site/
├── index.html          # Главная страница
├── styles.css          # Стили (Desktop-first подход)
├── script.js           # JavaScript для интерактивности
├── fonts/              # Локальные шрифты
│   └── README.md       # Инструкция по установке шрифтов
├── images/             # Изображения
│   ├── images-png/     # PNG стикеры для декора
│   └── images-svg/     # SVG логотипы
└── README.md           # Документация
```

## Структура сайта

1. **Header** - Логотип команды AIX с названием
2. **Фото команды** - Общее фото команды
3. **Участники команды** - Карточки с информацией о каждом участнике (8 участников + 1 тренер)
4. **Фото робота** - Фото робота команды в текущем сезоне
5. **Footer** - Контакты команды

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server
```

Затем откройте `http://localhost:8000` в браузере.

## Настройка контента

### Добавление фотографий

1. **Фото команды**: `images/team-photo.jpg`
2. **Фото робота**: `images/robot-photo.jpg`
3. **Фото тренера**: `images/coach.jpg`
4. **Фото участников**: `images/member1.jpg` до `images/member8.jpg`

### Заполнение информации

Откройте `index.html` и обновите:
- Имена и фамилии участников
- Роли в команде
- Контакты (email и Telegram)
- Контакты команды в footer

## Декоративные элементы

На заднем фоне используются PNG стикеры из папки `images/images-png/`:
- 9 декоративных элементов
- Автоматическая адаптация размера для разных устройств
- Низкая прозрачность для ненавязчивого фона

## Технические детали

- Чистый HTML, CSS и JavaScript (без фреймворков)
- CSS переменные для цветов бренда
- Intersection Observer для анимаций при скролле
- Оптимизированные изображения с placeholder'ами
- Поддержка prefers-reduced-motion для доступности
