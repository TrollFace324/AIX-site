# AIX - Калькулятор себестоимости 3D печати

Современный веб-сайт для команды AIX с калькулятором себестоимости 3D печати.

## Структура проекта

```
AIX-site/
├── index.html      # Главная страница
├── styles.css      # Стили (Desktop-first подход)
├── script.js       # JavaScript для интерактивности
└── README.md       # Документация
```

## Брендинг

### Цветовая палитра
- **Неоновый зеленый**: `#AAFF16`
- **Светло-серый/белый**: `#EBE9F6`
- **Черный/темно-серый**: `#100918`
- **Темно-фиолетовый**: `#411774`

### Шрифты
- **Основной**: Anton (Google Fonts)
- **Вторичный**: Системные шрифты (San Francisco, Segoe UI, Roboto и др.)

## Особенности

- ✅ Адаптивный дизайн (Desktop-first подход)
- ✅ Современная типографика
- ✅ Плавные анимации и переходы
- ✅ Доступность (accessibility)
- ✅ Оптимизация производительности
- ✅ Семантическая HTML разметка

## Адаптивность

Сайт оптимизирован для:
- **Desktop**: 1024px и выше
- **Tablet**: 768px - 1024px
- **Mobile**: 480px - 768px
- **Small Mobile**: до 480px

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server
```

Затем откройте `http://localhost:8000` в браузере.
