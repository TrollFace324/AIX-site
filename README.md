# AIX - Команда по робототехнике

Современный веб-сайт команды AIX по робототехнике.

## Структура проекта

```
AIX-site/
├── index.html              # Главная страница
├── script.js               # JavaScript для интерактивности
├── styles/                 # CSS файлы
│   ├── variables.css      # CSS переменные (цвета, шрифты)
│   ├── globals.css        # Сброс стилей и базовые настройки
│   ├── animations.css     # Анимации и переходы
│   └── styles.css         # Основные стили компонентов
├── fonts/                  # Локальные шрифты (WOFF2)
│   ├── fonts.css          # Подключение шрифтов (@font-face)
│   ├── anton-regular.woff2 # Шрифт Anton (WOFF2)
│   └── README.md          # Инструкция по установке шрифтов
├── images/                 # Изображения
│   ├── images-png/        # PNG стикеры для декора
│   └── images-svg/        # SVG логотипы
└── README.md               # Документация
```

## Структура сайта

1. **Header** - Логотип команды AIX с названием
2. **Фото команды** - Общее фото команды
3. **Участники команды** - Карточки с информацией о каждом участнике (8 участников + 1 тренер)
4. **Фото робота** - Фото робота команды в текущем сезоне
5. **Footer** - Контакты команды

## Организация CSS

Проект использует модульную структуру CSS:

- **`fonts/fonts.css`** - Подключение шрифтов через `@font-face` (формат WOFF2)
- **`styles/variables.css`** - CSS переменные для цветов бренда и шрифтов
- **`styles/globals.css`** - Сброс стилей браузера, базовые настройки, доступность
- **`styles/animations.css`** - Все анимации и переходы (hover, transitions)
- **`styles/styles.css`** - Основные стили компонентов и адаптивный дизайн

**Порядок подключения в HTML:**
1. `fonts/fonts.css` - сначала шрифты
2. `styles/variables.css` - затем переменные
3. `styles/globals.css` - базовые стили
4. `styles/animations.css` - анимации
5. `styles/styles.css` - основные стили

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server
```

Затем откройте `http://localhost:8000` в браузере.
